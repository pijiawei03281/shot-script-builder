Shot Script Builder is a visual planning tool for directors, cinematographers, and creators. 
It supports 2D and 3D blocking, camera movement design, lighting setup planning, reference boards, scene management, shooting schedules, call sheets, storyboard exports,
and AI prompt preparation.

Shot Script Builder 是一個視覺化鏡頭腳本與分鏡規劃工具，支援 2D/3D blocking、運鏡設計、燈光配置、參考圖管理、場景管理、拍攝排程、通告單、分鏡匯出與 AI prompt 準備。
